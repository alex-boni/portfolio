<a name="design"></a>  
## 🖥️ Design  
The portfolio follows a minimalist and modern design, with a responsive layout that adapts to different devices.
Each section (About, Projects, Education, Experience, Skills, Certifications) is accessible from the navbar with smooth scrolling.
Tailwind CSS is used to ensure a clean, scalable, and accessible UI.

<a name="installation"></a>  
## ⚙️ Installation  

1. Clone the repository:  
   ```bash
   git clone https://github.com/alex-boni/portfolio.git
   ```
2. Move into the project folder:  
   ```bash
   cd portfolio
   ```
3. Install dependencies:  
   ```bash
   npm install
   ```
4. Run the app locally:  
   ```bash
   npm run preview
   ```

---

### ⚙️ Or Create a New React + Vite App from Scratch  

1. Create a React application called `portfolio`:  
   ```bash
   npm create vite@latest portfolio -- --template react 
   ```
2. Move into the project folder:  
   ```bash
   cd portfolio
   ```
3. Install dependencies:  
   ```bash
   npm install
   ```
4. Install Tailwind CSS v4:  
   ```bash
   npm install tailwindcss @tailwindcss/vite
   ```
5. Update scripts in `package.json`:  
   ```bash
   npm pkg set scripts.lint="eslint . --ext js,jsx --report-unused-disable-directives --max-warnings 0" 
   npm pkg set scripts.preview="vite build && vite preview --host"
   ```
6. Run the app:  
   ```bash
   npm run preview
   ```

<a name="usage"></a>  
## 📌 Usage  

1. **Hero / Landing**: The site opens with an engaging hero section—“Ey Devs! Soy Alex!”—with subtle animations to draw the visitor in (Coming soon animations).
2. **Navigation**: The sticky navbar allows you to jump to each section smoothly: About, Projects, Education, Experience, Skills, Certifications, Contact.
3. **About Me**: Personal intro and photo, with option to download resume(Coming soon) or view LinkedIn.
4. **Projects**: Interactive cards show project overview, tech stack, and quick links (“Live” / “Code”), with hover effects adding visual depth.
5. **Education & Experience**: Clean vertical chronological layout displaying your academic and work journey.
6. **Skills**: Hard skills and soft skills can include details with XYZ structure.
7. **Certifications**: Each certificate is shown with an image; clicking “Ver más” reveals details or "Ver certificado" navigates to the cert link.
8. **Contact Section**: End with a contact form and icons linking to email and LinkedIn reach for recruiters (Coming soon available form).

This structured flow ensures that visitors grasp your story, skills, and work in a few scrolls, and encourages interaction at each step.

<a name="uso"></a>  
## 📌 Uso  

1. **Hero / Landing**: La página se abre con una sección de bienvenida atractiva —“Ey Devs! Soy Alex!”— con animaciones sutiles que captan la atención del visitante (Proximamente disponibles animaciones).  
2. **Navegación**: La barra de navegación fija permite saltar suavemente entre secciones: Sobre mí, Proyectos, Educación, Experiencia, Habilidades, Certificaciones y Contacto.  
3. **Sobre mí**: Introducción personal y foto, con opción de descargar el CV (proximamente disponible) o visitar mis redes sociales.  
4. **Proyectos**: Tarjetas interactivas muestran un resumen del proyecto, stack tecnológico y enlaces rápidos (“Live” / “Code”), con efectos hover que añaden profundidad visual.  
5. **Educación & Experiencia**: Diseño limpio y cronológico en vertical que muestra tu recorrido académico y profesional.  
6. **Habilidades**: Las hard skills y las soft skills pueden incluir tooltips o pequeños modales con más detalles.  
7. **Certificaciones**: Cada certificado se muestra con una imagen; al hacer clic en “Ver más” se despliegan los detalles y en "Ver certificado" se abre el enlace al certificado.  
8. **Contacto**: Otra vista que ofrece un formulario de contacto y enlaces directos a email y LinkedIn sencillo para reclutadores (Proximamente disponible el formulario).  

Este flujo estructurado garantiza que los visitantes comprendan tu historia, habilidades y proyectos en pocos scrolls, y fomenta la interacción en cada paso.
